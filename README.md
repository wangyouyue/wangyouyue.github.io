# Chongzhi Yin — Academic Profile

Static source for <https://wangyouyue.github.io/>.

The website is a self-contained, responsive HTML page with English, Simplified Chinese, Traditional Chinese, Japanese, Korean, and Arabic content. It includes selected publications, research interests, academic experience, honors, public researcher profiles, a public CV link, institutional contact information, system-aware appearance modes, and accessible navigation.

## Publishing

GitHub Pages serves the files in the `main` branch root. Future updates normally require replacing `index.html`; visual and media changes may also update `assets/`, `cloud-background.png`, or `og.png`.
