# Chongzhi Yin — Academic Profile

Static source for <https://wangyouyue.github.io/>.

The website is a self-contained, responsive HTML page with English, Chinese, and Japanese content. It includes selected publications, research interests, academic experience, honors, public researcher profiles, a public CV link, and institutional contact information.

## Publishing

GitHub Pages serves the files in the `main` branch root. Future updates normally require replacing `index.html`; visual asset changes may also update `cloud-background.png` or `og.png`.
