# Icon sources

- Brand marks (`orcid.svg`, `googlescholar.svg`, `researchgate.svg`, `github.svg`): [Simple Icons](https://github.com/simple-icons/simple-icons), CC0-1.0.
- Interface symbols (`house.svg`, `microscope.svg`, `user-round.svg`, `link-2.svg`, `mail.svg`, `monitor.svg`, `sun.svg`, `moon.svg`, `file-text.svg`, `external-link.svg`, `arrow-up.svg`): [Lucide](https://github.com/lucide-icons/lucide), ISC License.

The site embeds these marks as monochrome CSS masks so their color follows the active light or dark theme.
